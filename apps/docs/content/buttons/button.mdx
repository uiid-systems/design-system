## Features & Tips

- Polymorphic `render` prop can be used to compose a button with a custom trigger.
- Compose with an anchor or a custom `Link` component by setting the `nativeButton` prop to `false`.

## Accessibility

- Built on Base UI Button with proper ARIA attributes
- Keyboard: `Enter` and `Space` trigger click
- Supports `aria-label` for icon-only buttons
- `disabled` state is properly announced
