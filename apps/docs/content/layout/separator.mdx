A visual divider built on [Base UI Separator](https://base-ui.com/react/components/separator). Use it to break up sections within a <ComponentLink name="Stack" /> or <ComponentLink name="Group" />.

### Quick tips

- Defaults to `horizontal` orientation. Set `orientation="vertical"` for side-by-side layouts.
- Use the `shade` prop to match the separator color to your surface context.
