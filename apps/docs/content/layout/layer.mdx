A container for overlapping elements. Layer builds on <ComponentLink name="Box" /> and adds an `offset` prop that shifts each child by a calculated amount, useful for stacked avatars, fanned cards, or any overlapping layout.

### Quick tips

- Pass `offset={{ x: number, y: number }}` to shift each child incrementally using CSS transforms.
- Layer automatically adds padding to account for shifted children so nothing gets clipped.
- Without an `offset`, Layer behaves like a standard <ComponentLink name="Box" />.
