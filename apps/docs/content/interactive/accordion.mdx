Collapsible accordion panels for organizing content into expandable sections, built on [Base UI Accordion](https://base-ui.com/react/components/accordion).

### Quick tips

- Pass an array of `items` with `value`, `trigger`, and `content` to render panels.
- Use `multiple` to allow more than one panel open at a time.
- Set `defaultValue` to control which panels start expanded.
- Fine-tune internals with `RootProps`, `TriggerProps`, and `PanelProps`.
