Breadcrumb navigation showing page hierarchy. Renders a list of links with separators to indicate the user's current location within a site structure.

### Quick tips

- Pass an array of `items` with `label` and optional `href` to render the breadcrumb trail.
- The last item is automatically rendered as the current page (no link).
- Use the `separator` prop to customize the divider between items.
