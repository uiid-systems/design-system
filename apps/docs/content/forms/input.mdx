A single-line text field built on [Base UI Input](https://base-ui.com/react/components/input). Input handles label association, description text, and error states out of the box when you provide a `label` or `description` prop â€” otherwise it renders as a plain input with no wrapper.

### Quick tips

- Comes in three sizes: `small`, `medium`, and `large`. Defaults to `medium`.
- Use `fullwidth` to stretch the input to fill its container.
- The `ghost` variant strips visual chrome for inline or embedded use cases.
- Pass `FieldProps` to customize the label and description wrapper independently.

## Accessibility

- Label is automatically associated with the input via `id`
- Description text is linked with `aria-describedby`
- Supports `required`, `disabled`, and `readOnly` states
- Error states are announced to screen readers
