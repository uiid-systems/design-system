Numeric input with increment/decrement buttons and optional label, built on [Base UI NumberField](https://base-ui.com/react/components/number-field).

### Quick tips

- Set `min` and `max` to constrain the value range.
- Use `step` to control increment/decrement granularity.
- Supports `label` and `description` props for form field labeling.
- Comes in three sizes: `small`, `medium`, `large`.
