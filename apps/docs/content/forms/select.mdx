A dropdown select built on [Base UI Select](https://base-ui.com/react/components/select). Pass an `items` array for simple use cases, or use children for fully custom option rendering. Select is generic — it accepts any value type, not just strings.

### Quick tips

- The trigger inherits <ComponentLink name="Input" /> styling, so `size`, `fullwidth`, and `ghost` variants work here too.
- Items support `label`, `value`, `disabled`, `description`, and `icon` properties.
- The dropdown renders through a portal, so it won't be clipped by overflow containers.
- Like other form components, providing a `label` or `description` will automatically wrap the trigger in a Field.
- Fine-tune internals with `RootProps`, `TriggerProps`, `PortalProps`, `PositionerProps`, `PopupProps`, `ListProps`, `ValueProps`, `IndicatorProps`, and `FieldProps` — or import `SelectRoot`, `SelectTrigger`, `SelectPortal`, `SelectPositioner`, `SelectPopup`, `SelectList`, `SelectValue`, `SelectIndicator`, and `SelectItem` directly for full control over composition.
