Radio button input with label and description support, built on [Base UI Radio](https://base-ui.com/react/components/radio-group). Use inside a <ComponentLink name="RadioGroup" /> for single-select behavior.

### Quick tips

- Always use within a <ComponentLink name="RadioGroup" /> for proper single-select semantics.
- Supports `label` and `description` props for field labeling.
- Use `disabled` to prevent interaction on individual options.
