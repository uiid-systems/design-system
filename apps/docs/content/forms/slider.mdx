Range slider input with optional label and description, built on [Base UI Slider](https://base-ui.com/react/components/slider).

### Quick tips

- Set `min`, `max`, and `step` to control the slider range and granularity.
- Supports `label` and `description` props for form field labeling.
- Use `disabled` to prevent interaction.
