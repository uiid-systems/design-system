Progress bar indicator for showing completion status. Renders a horizontal bar that fills based on the current value.

### Quick tips

- Set `value` as a number between 0 and 100 to control the fill level.
- Use `tone` to apply semantic coloring (`positive`, `critical`, `warning`, `info`).
- Supports `label` for accessible labeling.
